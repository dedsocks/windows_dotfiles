# Setup Guide

### 1. Install Required Applications

Before using the configuration files, install the following:

* Komorebi
* YASB (Yet Another Status Bar)
* Windhawk
* PowerToys (for improved search)
* Cava

---

### 2. Apply Configuration Files

Copy the provided configuration files into their respective directories:

* **Komorebi** → place `komorebi.json` in your home folder
* **YASB** → place `config.yaml` and `styles.css` in `~/.config/yasb/`
* **Windhawk** → import or apply the provided settings

---

### 3. Terminal & Power Settings

#### Accessing Terminal Settings

* Open Windows PowerShell
* Right-click on the title bar
* Click **Settings**

#### Adjusting Transparency

* In the left sidebar, select **Windows PowerShell**
* Go to **Appearance**
* Enable transparency
* Set opacity 
* Enable **Acrylic material**

#### Power Settings

* Windows key + A (Open action center)
* Turn **off power saver mode**

Note: Transparency effects may not work properly if power saver mode is enabled.

---

### 4. Improved Search

Using PowerToys:

* Enable **PowerToys Run**
* Use `Alt + Space` to open it
* This replaces Windows search and removes unnecessary results
