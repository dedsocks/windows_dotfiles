# Windhawk Mods

Mods installed via [Windhawk](https://windhawk.net) for this setup.

## 1. Taskbar Height and Icon Size

Reduces taskbar size and gets rid of all the problems with the default auto hide taskbar setting.

| Setting | Value |
|---|---|
| Icon size | 1 (small) |
| All other fields | 0 |

> **Note:** Setting small icon size to 0 causes an error — keep it at 1 as the minimum.

## 2. Translucent Windows

Enables acrylic/blur transparency on Win32 windows system-wide.
